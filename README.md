COLLEGE ADMISSION SYSTEM (C++)

ABOUT THE PROJECT
This is a simple console based College Admission System developed using C++.
The purpose of the project is to manage basic college information like college name, address, total strength and branches offered.

The project was created as part of our Object Oriented Programming (OOP) coursework in second year.

The system stores college data using binary file handling so the information can be saved and accessed later.

---

FEATURES

The program provides the following options in the menu:

1. Register a New College
   Allows the user to enter the college name, address and total strength.

2. Edit College Strength
   The total strength of the college can be modified.

3. Add New Branch
   Branches can be added along with their individual strength.

4. View College Details
   Displays the college information and the branches available.

5. Exit Program

---

CONCEPTS USED

This project uses some important C++ concepts such as:

* Classes and Objects
* Constructors
* File Handling using binary files
* Arrays and strings
* Menu driven programming
* Basic console interface

---

DATA STORAGE

Each college is stored in a binary file using the college name as the file name.

The program writes and reads the college object directly into the file using file handling.
This allows the data to remain saved even after the program closes.
